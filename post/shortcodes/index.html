Music 163

Params

  • id

    • required param
    • you can extract from music url
    • url format http://music.163.com/#/song?id=28196554

  • Fiddle auto

    • optional param
    • default value 0
    • you can overwrite it with 1

Examples

  • Simple
1
2

{{% music "28196554" %}}
{{% music "28196554" "1" %}}
